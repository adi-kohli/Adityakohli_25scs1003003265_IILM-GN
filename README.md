🌾 Pollinator Decline & Crop Yield Prediction (AI Project)

📌 Overview

This project analyzes the impact of pollinator decline on agricultural crop yield using Machine Learning.

It integrates multiple environmental and agricultural datasets to:

📊 Study relationships between pollinators and crop yield

🌍 Analyze effects of rainfall, pesticides, and temperature

🤖 Predict crop yield using Machine Learning models

🚀 Features

✅ Multi-dataset integration (5 CSV files)

✅ Data cleaning & preprocessing

✅ Dataset merging using common keys (area + year)

✅ Machine Learning model implementation

✅ Prediction based on real-world features

✅ Performance evaluation using RMSE and R²

📂 Dataset Used

The project uses the following datasets:

1. pesticides.csv – Pesticide usage data
2. rainfall.csv – Rainfall data
3. US_honey_dataset_updated.csv – Honey bee colonies data
4. yield_df.csv – Crop yield dataset
5. pollinators_dataset_100rows.csv – Pollinator dataset

🧠 Machine Learning Workflow

1. Data Cleaning
2. Column Standardization
3. Dataset Merging (based on area + year)
4. Handling Missing Values
5. Feature Selection
6. Model Training (Linear Regression)
7. Model Evaluation
8. Prediction

📈 Model Performance

RMSE ≈ 60 (acceptable due to multi-source dataset variations)

R² Score: varies depending on data
